# Instalación de VITE

## Requisitos previos
- Tener instalado [Node.js](https://nodejs.org/en/download) y npm.

## Instalación de Vite

Abre una terminal y ejecuta:

```bash
npm create vite@latest nombre-del-proyecto -- --template react
```

Sigue las instrucciones para configurar tu proyecto.

## Instalación de dependencias

```bash
cd nombre-del-proyecto
npm install
```

## Ejecución del proyecto

```bash
npm run dev
```

Esto iniciará el servidor de desarrollo. Abre el navegador en la URL que aparece en la terminal.

# Tambien se puede realizar de la siguiente manera
Abre una terminal y ejecuta:

```bash
npm create vite@latest
```
